# E-Learning Client (React + Vite)

## Setup
1. Create `.env` with `VITE_API_URL=http://localhost:4000/api` (optional).
2. Install deps: `npm install`
3. Run: `npm run dev`

## Pages
- Courses list
- Course detail with video streaming + PDF link
- Quiz taking page
- Dashboard (enrollments + quiz attempts)
- Login
